🗳️ ElectraGuide (VoxAgent)
The Agentic "Truth Engine" for the 2026 Global Election Cycle
ElectraGuide is a decentralized Civic OS designed to safeguard election integrity through real-time bias auditing, cryptographic security simulations, and multi-national voter intelligence. Built using a "Vibe Coding" architecture, it leverages a massive 25,000-character reasoning framework to provide unbiased, high-fidelity civic assistance.

🚀 LIVE DEMO: View the Production App
✨ Key Features
🛡️ Media Shield (Bias Analyzer): Real-time NLP analysis of news headlines using an Unbiased Reasoning Loop to detect proxy variables and loaded language.

🔐 Election Security Lab: Interactive simulation of M3-EVM "Crypto-Handshake" protocols to educate voters on backend security.

🌍 Multi-National Intelligence: Context-aware voter rights data for the US, UK, and India, delivered through a persona-driven UX.

🧩 Simple Mode: A one-toggle accessibility feature that adapts technical jargon for universal voter comprehension (WCAG 2.1 Compliant).

🧠 The Brain: 25k-Character Agentic Logic
Unlike traditional apps, ElectraGuide’s core logic is powered by a high-density Reasoning Loop (detailed in GEMINI.md). This framework allows the AI to:

Remediate Bias before outputting data.

Attach Fairness Scorecards to every civic interaction.

Simulate Edge-Cases in election law across different jurisdictions.

🛠️ Tech Stack
Frontend: Next.js 14+ (App Router, Tailwind CSS, TypeScript)

Backend: Spring Boot (Java 21, Hexagonal Architecture)

Database: PostgreSQL (Cloud SQL ready)

Deployment: Google Cloud Run (Fully Containerized via Docker)

AI Engine: Gemini 3.1 Pro (via Google Antigravity)

📂 Project Structure
/frontend: Next.js web application.

/backend: Spring Boot REST API.

/artifacts: Technical deep-dives, "Vibe Coding" strategy, and social audit reports.

GEMINI.md: The 25,000-character master logic framework.

🚀 Getting Started
Local Development
